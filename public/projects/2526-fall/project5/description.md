
# Waste Sorting Assistant

Computer-vision demo that classifies household waste into common recycling categories. The project contains a small dataset, a browser demo, and a simple heuristic to suggest correct bins.

Features

- On-device image classification demo (browser)
- Example dataset with labeled images
- Simple confidence-based suggestions and fallback instructions
- Exportable results for evaluation

Tech stack

- React + TensorFlow.js for the client demo
- Small Python script to prepare the example dataset

Try it

Open the demo and upload or drag an image to see the predicted category and confidence score.
