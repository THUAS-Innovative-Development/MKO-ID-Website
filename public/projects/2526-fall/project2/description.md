# Urban Mobility Mapper

Interactive map for visualising urban mobility patterns using real-time transport and GPS datasets. The app showcases travel demand heatmaps and route clustering to help city planners explore mobility hotspots.

Features

- Map-based clustering of trips and origin/destination flows
- Route heatmaps and time-of-day filtering
- CSV/GeoJSON export for selected regions
- Simple UI for comparing days and modes of transport

Tech stack

- React, Leaflet for mapping
- Turf.js for spatial analysis
- Lightweight Node service to serve sample trip data

Try it

Use the time slider to play mobility patterns through a day and inspect cluster popups for sample trip counts.
